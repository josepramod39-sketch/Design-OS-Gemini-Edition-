# Tailwind Color Configuration

## Color Choices

- **Primary:** `violet` — Used for buttons, links, key accents
- **Secondary:** `slate` — Used for tags, highlights, secondary elements
- **Neutral:** `zinc` — Used for backgrounds, text, borders

## Usage Examples

Primary button: `bg-violet-600 hover:bg-violet-700 text-white`
Secondary badge: `bg-slate-100 text-slate-800`
Neutral text: `text-zinc-600 dark:text-zinc-400`
