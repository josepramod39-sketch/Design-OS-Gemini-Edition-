# EdithRp — Product Overview

## Summary

An intelligent backend inventory management system designed for liquor stores, using Gemini AI to automate stock tracking via invoice scanning and provide predictive insights to optimize business performance.

## Planned Sections

1. **Inventory & Scanning** — Manage stock levels and automatically update inventory by scanning invoices/receipts.
2. **Analytics Dashboard** — Visualize sales trends, stock turnover, and performance reports.
3. **Predictive Insights** — View AI-driven suggestions for reordering stock and applying discounts to slow-moving items.
4. **Voice Command Center** — Search for products and query stock status using natural voice commands.

## Data Model

- Supplier
- Product
- Invoice
- InvoiceItem
- Inventory
- Insight

## Implementation Sequence

Build this product in milestones:

1. **Foundation** — Set up design tokens, data model types, and application shell
2. **Inventory & Scanning** — Core module for tracking stock and processing intake.
3. **Analytics Dashboard** — Comprehensive dashboard providing visual insights into business performance.
4. **Predictive Insights** — Intelligent "feed" of actionable business recommendations.
5. **Voice Command Center** — Sophisticated natural language interface for querying stock and sales.

Each milestone has a dedicated instruction document in `product-plan/instructions/`.
